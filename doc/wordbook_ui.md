# Page 1

## 화면 구성

세로 스크롤 없는 단일 화면. 단어 버튼 5개 + 하단 네비게이션.

## 단어 버튼 (× 5)

- 한 줄짜리 버튼이 세로로 5개 나열
- 기본 상태: 한자 표시 (예: 学生)
- 탭하면 토글되어 음독(한글) + 뜻으로 전환 (예: 가쿠세이 / 학생)
- 다시 탭하면 한자로 복귀

## 체크박스

- 각 단어 버튼 우측에 체크박스 배치
- 맞춘 단어를 체크하는 용도

## 하단 네비게이션

- 좌측: [다음] 버튼 — 다음 5개 단어로 이동
- 우측: 설정 아이콘(⚙️)

# Page2 (설정 아이콘을 클릭했을 때)

## 화면 구성

단어 입력 영역(상단)과 단어 리스트(하단)로 구분.

## 상단 바

- 좌측: ← 돌아가기 아이콘 (메인 화면으로 복귀)
- 우측: 저장 버튼

## 입력 영역

세 개의 input 필드가 세로로 나열:

1. 단어 (한자 입력)
2. 음독 (한글 표기 입력)
3. 뜻 (한글 의미 입력)

## 단어 리스트

- 구분선 아래로 전체 등록 단어가 세로로 나열
- 각 아이템 형식: 한자 + (음독) 뜻 + ✕ 삭제 아이콘
    - 예: 学生  (가쿠세이) 학생 ✕
- ✕ 아이콘 탭 시 해당 단어 삭제
- 리스트는 스크롤 가능
